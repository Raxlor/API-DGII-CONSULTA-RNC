API DGII República Dominicana: Consulta RNC y Cédula en tiempo real

Esta API permite consultar RNC y cédulas en la República Dominicana utilizando datos públicos de la DGII. Está diseñada para desarrolladores, sistemas SaaS, fintechs y plataformas contables que necesitan validar contribuyentes en tiempo real sin procesos manuales.

¿Qué es un RNC en República Dominicana?

El Registro Nacional de Contribuyentes (RNC) es el identificador fiscal utilizado por la DGII para registrar empresas y personas físicas con actividad económica. Este identificador es obligatorio para facturación, impuestos y validaciones comerciales.

Cómo consultar un RNC usando una API

Puedes consultar un RNC directamente mediante solicitudes HTTP utilizando endpoints como /api/v1/rnc/{rnc} o realizar búsquedas por nombre con /api/v1/buscar?q=texto. Esto permite integrar validaciones automáticas en sistemas ERP, CRM o plataformas de facturación electrónica.

Casos de uso de la API DGII

  • Validación automática de clientes en sistemas SaaS
  • Integración en plataformas de facturación electrónica
  • Verificación de proveedores y empresas
  • Automatización de procesos contables

Ventajas de usar esta API

  • Consulta RNC en tiempo real
  • Sin necesidad de registro

  • Optimizada para grandes volúmenes de datos

Preguntas frecuentes sobre la API RNC

Como consultar RNC y cedula por API en Republica Dominicana?

Usa el endpoint /api/v1/rnc/{rnc} para consultas directas y /api/v1/buscar?q=texto para coincidencias por razon social.

Necesito registro para usar la API DGII?

No. Puedes integrar la API sin crear cuenta, respetando el limite de uso para mantener estabilidad del servicio.
