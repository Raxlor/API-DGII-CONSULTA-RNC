# 🇩🇴 Consulta RNC – API Abierta República Dominicana

![Node.js](https://img.shields.io/badge/Node.js-18+-green)
![License](https://img.shields.io/badge/License-MIT-blue)

Una API REST sencilla, eficiente y abierta para consultar información pública de contribuyentes (RNC) en la República Dominicana.

Este proyecto nace con el objetivo de ofrecer una alternativa limpia, gratuita y transparente para desarrolladores que necesiten validar datos tributarios de forma rápida y confiable.

---

## 🌟 Filosofía del Proyecto

- ✅ Código abierto y auditable
- ✅ Enfoque en simplicidad y rendimiento
- ✅ Sin restricciones complejas de uso
- ✅ Orientado a la comunidad dominicana

Creemos en herramientas que reduzcan fricción técnica y aceleren el desarrollo de soluciones locales.

---

## 🕒 Datos Siempre Actualizados

El valor diferencial de este nodo es la frescura de la información:

- 🔄 Actualización semanal automática (cada 7 días)
- 📊 Base de datos optimizada para consultas rápidas
- 🎯 Validaciones alineadas con la realidad tributaria vigente

---

## 🚀 Cómo Empezar

### 🌐 Uso Directo (Web)

Puedes realizar consultas manuales y probar la API desde el portal oficial:

👉 [api-dgii.dominicantechnology.com](https://api-dgii.dominicantechnology.com/)

---

### 💻 Desarrollo Local

Si deseas estudiar el código o contribuir:

```bash
# 1️⃣ Clonar repositorio
git clone [https://github.com/Raxlor/API-RNC-DGII](https://github.com/Raxlor/API-RNC-DGII)

# 2️⃣ Entrar al proyecto
cd API-RNC-DGII

# 3️⃣ Instalar dependencias
npm install

# 4️⃣ Configurar variables de entorno (Crea tu propio .env)
# Revisa el archivo .gitignore para asegurar que tus credenciales estén protegidas.

# 5️⃣ Ejecutar
node server.js
